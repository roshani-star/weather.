# 🌤 Weather App

A modern Weather Application built using **Python, Flask, OpenWeather API, HTML, and CSS**. Users can search for any city and instantly get live weather information with a clean and responsive interface.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡 Current temperature
- 🤗 Feels like temperature
- 💧 Humidity
- 💨 Wind speed
- 🌎 Country information
- 🎨 Modern Glassmorphism Design

---

## 🛠 Tech Stack

- Python
- Flask
- HTML5
- CSS3
- OpenWeather API

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### 2. Go to project folder

```bash
cd weather-app
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```env
API_KEY=YOUR_OPENWEATHER_API_KEY
```

### 5. Run the application

```bash
python app.py
```
